# CliqRise Marketing Website

**The missing link between ad spend and affiliate earnings.**

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS + shadcn/ui
- **Theme:** Starry Night (TweakCN) - Dark navy with gold accents
- **Animations:** Framer Motion
- **Typography:** Libre Baskerville (headings) + Inter (body)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Design System

See `/design-system/` for:
- `cliqrise-design-system.json` - Complete design tokens
- `starry-night-theme.css` - TweakCN theme + custom styles
- `CLINE.md` - Project context for AI assistants
- `CLINE-PROMPTS.md` - Step-by-step build prompts
